# 🌿 Wall Calendar App

A modern, aesthetic **wall calendar web application** built with Next.js that replicates the feel of a physical wall calendar while offering powerful interactive features like date range selection, notes, and theme switching.

---

## 🌐 Live Demo

🚀 **Try it here:**
https://wall-calendar-aryan.vercel.app/

---

## ✨ Features

### 📅 Wall Calendar UI

* Inspired by real-world wall calendars
* Hero image + calendar grid layout
* Clean and visually balanced design

### 🔵 Date Range Selection

* Select start and end dates
* Smooth visual highlighting
* Clear distinction between:

  * Start date
  * End date
  * In-between range

### 📝 Notes System

* Add notes for selected dates or ranges
* Edit and delete notes
* Month-based note management

### 🌗 Dark / Light Mode

* Smooth theme toggle
* Persistent user preference
* Fully styled for both modes

### 📱 Fully Responsive

* **Desktop:** Split layout (hero + calendar)
* **Mobile:** Stacked layout with seamless UX

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Aryan-2076/wall-calendar-app.git
cd wall-calendar-app
```

---

### 2. Install dependencies

```bash
npm install
```

---

### 3. Run the development server

```bash
npm run dev
```

Open: http://localhost:3000
